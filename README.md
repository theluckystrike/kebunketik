# Kebun Ketik

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

> A typing game for kids to practice typing by defeating pests with words.

Kebun Ketik is an engaging typing game designed for children to practice and improve their typing skills. Players defeat pests by typing words correctly, progressing through multiple stages with increasing difficulty and challenging boss fights.

## How to Play

1. Open `index.html` in a web browser
2. Type the words that appear on screen to defeat the pests
3. Complete each stage by typing all words correctly
4. Face challenging bosses at the end of each level
5. Progress through all stages to win!

## Features

- **Fun Learning** - Practice typing while playing an exciting game
- **Multiple Stages** - Progress through various levels of increasing difficulty
- **Boss Fights** - Challengeboss battles at the end of each stage
- **Kid-Friendly** - Designed for children with simple, intuitive controls
- **No Installation** - Runs directly in the browser

## Project Structure

```
├── index.html    # Main game file
├── style.css     # Game styles
└── script.js     # Game logic
```

## Getting Started

Simply open the game in any modern web browser:

```bash
# Using Python's built-in server
python3 -m http.server 8000

# Then open http://localhost:8000 in your browser
```

Or simply double-click `index.html` to open it directly.

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Keyboard for typing
