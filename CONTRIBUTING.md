# Contributing to kebunketik

Thank you for your interest in contributing to kebunketik!

## How to Report Bugs

1. **Check existing issues** - Search the [issue tracker](https://github.com/theluckystrike/kebunketik/issues) to see if your bug has already been reported
2. **Create a new issue** - Use the bug report template and include:
   - Clear description of the problem
   - Steps to reproduce the bug
   - Expected vs actual behavior
   - Any relevant error messages or screenshots

## How to Submit Pull Requests

1. **Fork the repository** and create a new branch from `main`
2. **Make your changes** - Ensure your code follows the project's style
3. **Submit a PR** - Describe what the change does and why it's needed

## Code Style Notes

- Use consistent indentation (2 spaces)
- Keep code clean and readable
- Add comments where helpful

## Development Setup

```bash
# Clone the repository
git clone https://github.com/theluckystrike/kebunketik.git
cd kebunketik

# Open index.html in your browser to test
```
